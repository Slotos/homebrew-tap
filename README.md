# Slotos Tap

## How do I install these formulae?

`brew install slotos/tap/<formula>`

Or `brew tap slotos/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "slotos/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
